Why RenLocalizer?

Designed to save you time and money.

Fully Automated

No more copy-pasting. RenLocalizer parses your Ren'Py script files and translates dialogue automatically.

Code Safe

It understands Ren'Py syntax. It ignores code blocks, variables, and tags, translating only what the player sees.

Privacy Focused
