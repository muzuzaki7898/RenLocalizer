Why RenLocalizer?

Designed specifically for Game Developers.

Fully Automated

No more copy-pasting. RenLocalizer parses your Ren'Py script files and translates dialogue automatically.

Code Safe

It understands Ren'Py syntax. It ignores code blocks, variables, and tags, translating only what the player sees.

Privacy & Local LLM

Don't want to send data to the cloud? Use Local LLM support (Ollama/LM Studio) to translate offline.

Multi-API Support

Use free Google Translate, high-quality DeepL, or powerful GPT-4/Gemini models. The choice is yours.
